# Car World - Open World Driving & Action Game

Car World is an open-world car simulation and action game where players can explore a large map, purchase vehicles, drive freely, and interact with the environment. The game includes an in-game admin system for managing players and game settings.

## Features

* Large open-world map to explore
* Buy and own multiple vehicles
* Realistic driving mechanics
* In-game admin access and management tools
* NPC system with combat interactions
* Vehicle dealership and economy system
* Player progression and rewards
* Customizable gameplay experience

## Gameplay

Players can earn money, purchase new cars, drive across the map, complete activities, and engage with NPCs throughout the world. Administrators have access to special commands and controls to manage the game and enhance the player experience.
